# XCEP
## About

XCEP is a library to search for CEP, it is the result of a tutorial on how to consume data from an external api 
## Installation

```elixir
def deps do
  [
    {:xcep, "~> 0.1.0"}
  ]
end
```

## Using

```elixir
  XCEP.cep "12345678"

```
